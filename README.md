##  Getting Started

Clone the repository, open its folder and install dependencies with:
```bash
// This application requires Node.js v16.13+.  
git clone https://github.com/twobanks/elumini_frontend.git
cd elumini_frontend
npm install
npm run start
```

Run tests
```bash
npm run test
```

Run build
```bash

npm run build

```
